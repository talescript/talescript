Design basics

Focus on mobile first. Designing with limited space means that you need to think about pages and interactions on a phone. By starting with the harder problem first, which is making something usable on a tiny screen, you then adopt a solution to the easier problem, the big screen.

Similarly, design with black and white first.

This constraint forces you to focus on spacing and laying out elements. Making it beautiful and usable in every way. Black and white forces you to focus on things like spacing, sizes, and layout first.

Adding one color to a grayscale site draws the eye easily.

Double your whitespace

If you want to make UI that looks designed, add a lot of breathing room. Add whitespace between text, elements and groups of elements. If you design from a blank canvas, you are adding in elements, with whitespace at the front.

Text on images

  • Add an overlay on every image
  • Text-in-a-box
  • Blur the image / blur part of the underlying image

Make text pop

If a site element needs emphasis, apply both up-pop and down-pop styles.
